## Awareness (ToFU)

*Kunden oppdager oss for første gang.*

### Kundens perspektiv

**Mål:**
- Forstå om dette er relevant for meg
- Få et førsteinntrykk av merkevaren

**Spørsmål de stiller seg:**
- Er dette en merkevare for folk som meg?
- Løser de et problem jeg har?
- Liker jeg stilen/tonen?

**Bekymringer:**
- Er dette til å stole på?
- Er det verdt tiden min å utforske videre?

### Touchpoints

**Betalte kanaler:**
- [ ] Facebook/Instagram Ads
- [ ] Google Ads
- [ ] Influencer-samarbeid
- [ ] Sponset innhold

**Organiske kanaler:**
- [ ] Sosiale medier (organisk)
- [ ] SEO / blogginnhold
- [ ] PR / medieomtale
- [ ] Word-of-mouth
- [ ] Fysiske lokasjoner

### Våre mål

- Fange oppmerksomhet (scroll-stoppers)
- Kommunisere kjernebudskap raskt
- Bygge initial tillit
- Trigge nysgjerrighet til å lære mer

### Relevante psykologi-prinsipper

- Mere Exposure Effect (gjentatt synlighet)
- Von Restorff Effect (skill deg ut)
- Picture Superiority (sterke bilder)
- Information Gap (skap nysgjerrighet)

## Consideration (MoFU)

*Kunden vurderer om vi er riktig valg.*

### Kundens perspektiv

**Mål:**
- Forstå hva vi tilbyr i dybden
- Vurdere om det passer deres behov
- Sammenligne med alternativer

**Spørsmål de stiller seg:**
- Hva er forskjellen mellom dere og konkurrentene?
- Passer dette mine verdier/preferanser?
- Hva sier andre kunder?

**Bekymringer:**
- Er det verdt prisen?
- Vil det fungere for meg?
- Kan jeg stole på det de sier?

### Touchpoints

**Digitale:**
- [ ] Nettside (kategorier, produktsider)
- [ ] Blogg / innholdsmarkedsføring
- [ ] YouTube / video
- [ ] Nyhetsbrev
- [ ] Retargeting-annonser

**Sosiale:**
- [ ] Reviews og omtaler
- [ ] Sosiale medier (engasjement)
- [ ] Community / grupper

### Våre mål

- Bevise brand-audience fit
- Bygge dypere tillit
- Utdanne om produktet/tjenesten
- Adressere vanlige innvendinger

### Relevante psykologi-prinsipper

- Social Proof (andres erfaringer)
- Authority (ekspertise og troverdighet)
- Confirmation Bias (bekreft deres tro)
- Halo Effect (kvalitetsoppfatning)

---

## Evaluation (BoFU)

*Kunden er klar til å velge, men trenger hjelp.*

### Kundens perspektiv

**Mål:**
- Finne riktig produkt/variant
- Få svar på praktiske spørsmål
- Føle seg trygg på valget

**Spørsmål de stiller seg:**
- Hvilken variant/størrelse passer meg?
- Hvordan fungerer retur/garanti?
- Hva sier reviews om dette spesifikke produktet?

**Bekymringer:**
- Hva om jeg velger feil?
- Hva om jeg angrer?
- Er dette den beste dealen?

### Touchpoints

**På nettsiden:**
- [ ] Produktsider med detaljer
- [ ] Størrelsesguide / produktvelger
- [ ] Reviews og ratings
- [ ] Sammenligningstabeller
- [ ] Live chat / kundeservice

**Remarketing:**
- [ ] Abandoned browse emails
- [ ] Retargeting ads med sett produkter

### Våre mål

- Gjøre det lett å finne riktig produkt
- Fjerne usikkerhet og risiko
- Skape urgency uten å pushe
- Tilby hjelp når de trenger det

### Relevante psykologi-prinsipper

- Decoy Effect (strategisk produktmiks)
- Scarcity (begrenset tilgjengelighet)
- Loss Aversion (hva de går glipp av)
- Choice Overload (ikke for mange valg)

## Purchase ($$)

*Kunden gjennomfører kjøpet.*

### Kundens perspektiv

**Mål:**
- Gjennomføre kjøp raskt og enkelt
- Føle seg trygg på transaksjonen

**Spørsmål de stiller seg:**
- Er betalingen sikker?
- Når får jeg produktet?
- Hva om noe går galt?

**Bekymringer:**
- Skjulte kostnader?
- Komplisert checkout?
- Spam etterpå?

### Touchpoints

- [ ] Handlekurv
- [ ] Checkout-prosess
- [ ] Betalingsløsning
- [ ] Ordrebekreftelse
- [ ] Live chat / support

### Våre mål

- Friksjonsfri checkout
- Bygge tillit i betalingsøyeblikket
- Minimere handlekurv-frafall
- Bekrefte at de tok riktig valg

### Relevante psykologi-prinsipper

- Default Options (forhåndsvalg)
- Goal Gradient (nesten ferdig!)
- Anchoring (vis originalverdi)
- Trust badges og garantier

---

## Post-purchase

*Tiden mellom kjøp og full verdi.*

### Kundens perspektiv

**Mål:**
- Få produktet raskt
- Bekrefte at de tok riktig valg
- Komme i gang med bruk

**Spørsmål de stiller seg:**
- Når kommer det?
- Gjorde jeg riktig valg?
- Hvordan får jeg mest ut av det?

**Bekymringer:**
- Kjøpsanger (buyer's remorse)
- Lever det opp til forventningene?
- Hva om det er feil/defekt?

### Touchpoints

- [ ] Fraktoppdateringer
- [ ] Leveringsopplevelse
- [ ] Unboxing
- [ ] Velkomst-epost / onboarding
- [ ] Brukerveiledning
- [ ] Kundeservice

### Våre mål

- Minimere kjøpsanger
- Skape wow-øyeblikk
- Hjelpe dem i gang
- Bygge grunnlag for relasjon

### Relevante psykologi-prinsipper

- Peak-End Rule (wow-øyeblikk + god avslutning)
- Cognitive Dissonance (bekreft valget)
- Endowment Effect (de eier det nå)
- Reciprocity (overrask med ekstra verdi)

## Loyalty (♻️)

*Fra kunde til gjenkjøper og ambassadør.*

### Kundens perspektiv

**Mål:**
- Få verdi av relasjonen
- Bli belønnet for lojalitet
- Holde seg oppdatert

**Spørsmål de stiller seg:**
- Er det verdt å kjøpe igjen?
- Hva får jeg ut av å være lojal?
- Hva er nytt?

**Bekymringer:**
- Spam og irrelevant kommunikasjon
- Føle seg utnyttet
- Bedre alternativer?

### Touchpoints

- [ ] Nyhetsbrev
- [ ] Lojalitetsprogram
- [ ] Eksklusive tilbud
- [ ] Community / grupper
- [ ] Sosiale medier
- [ ] Referral-program
- [ ] VIP-behandling

### Våre mål

- Øke gjenkjøpsrate
- Bygge emosjonell tilknytning
- Skape ambassadører
- Generere referrals

### Relevante psykologi-prinsipper

- Sunk Cost (investert i relasjonen)
- Social Identity (del av stammen)
- Ben Franklin Effect (be om hjelp)
- IKEA Effect (co-creation)
- Nostalgia (felles historie)

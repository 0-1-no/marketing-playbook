# Marketing Playbook - Sjekk Innhold

Verifiser at innhold samsvarer med prosjektets `BRAND.md`.

## Steg 1: Les BRAND.md

Finn og les `BRAND.md` fra prosjekt-rot eller nærmeste overordnede mappe.

Hvis ingen BRAND.md finnes:
```
⚠️ Ingen BRAND.md funnet i prosjektet.

Kjør `/marketing-playbook` for å sette opp.
```

## Steg 2: Identifiser innhold å sjekke

Spør brukeren eller sjekk kontekst for:
- Hvilken fil/tekst skal sjekkes?
- Er det en landing page, copy, email, ad, eller annet?

## Steg 3: ABC Sjekkliste

Evaluer innholdet mot hver seksjon:

### A - AUDIENCE

| Sjekk | Status | Kommentar |

| Relevant for målgruppen? | ✅/⚠️/❌ | [detaljer] |

## Steg 4: Spesifikke funn

### Ord som bør endres
List opp ord fra "Words We Avoid" som ble funnet:

| Funnet | Anbefalt endring |
|--------|------------------|
| "beste" | "pålitelig" / "effektiv" |

### Manglende elementer
List opp ting fra "Words We Use" eller "Key Messages" som kunne vært inkludert.

## Steg 5: Sammendrag

```
ABC Sjekkliste for [filnavn]

Score: [X/10]

✅ Styrker:
- [liste]

⚠️ Forbedringer:
- [liste med konkrete forslag]

❌ Kritisk:
- [hvis noe bryter sterkt med brand]
```

## Output-format

Vær konstruktiv og gi konkrete forbedringsforslag. Ikke bare påpek problemer - forklar hvordan det kan fikses.
